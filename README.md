## About

A simple browser-based memory card game where you flip cards to match pairs against the CPU.  
This project showcases the **core mechanic** from _Mosquintal: Memory_, a 3D commercial Unity project currently in development.

## Features

- Built with **Unity 6.3 LTS** and **C#**
- Simple game loop: title → play → match cards → end → restart

## Dependencies

- [DOTween](https://assetstore.unity.com/packages/tools/animation/dotween-hotween-v2-27676)

## How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/<elmonosapiens>/SimpleMemoryCardGame.git
   ```
2. Open the project in **Unity Hub** (Unity 6.3 LTS recommended).
3. Install **DOTween**:
   - Open **Unity Package Manager** → **My Assets**
   - Search for **DOTween (HOTween v2)** and install

4. Run the `Game` from the **Scenes** folder.
